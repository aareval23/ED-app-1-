# ED App 1

A simple local HTML/CSS/JavaScript application that you can download and run on your computer.

## How to Use

### Option 1: Download & Run Locally

1. **Clone or Download the Repository**
   - Click the green "Code" button on GitHub
   - Select "Download ZIP" to download all files
   - Extract the ZIP file to your computer

2. **Open the App**
   - Locate the `index.html` file in the extracted folder
   - Double-click `index.html` to open it in your default web browser
   - The app will load and be ready to use!

### Option 2: Using Git Command Line

```bash
git clone https://github.com/aareval23/ED-app-1-.git
cd ED-app-1-
```

Then open `index.html` in your browser.

## Customization

You can easily customize this app:

- **Edit HTML** (`index.html`) - Change the content and layout
- **Edit CSS** (`style.css`) - Modify colors, fonts, and design
- **Edit JavaScript** (`script.js`) - Add new features and functionality

Just save your changes and refresh the browser to see them!

## Requirements

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- No server or special software needed!

## Troubleshooting

**App won't open?**
- Make sure all files (index.html, style.css, script.js) are in the same folder
- Try right-clicking on `index.html` and selecting "Open with" > your favorite browser

**Changes not showing?**
- Try hard refresh: Press `Ctrl+Shift+R` (Windows) or `Cmd+Shift+R` (Mac)
